使用帮助 & 安装指南

1. 数据存储 (Local First)
本应用没有云端账号。所有数据存储在您选择的本地文件夹中（Journal/Inbox）。
要切换“账号”，只需点击左侧“连接本地工作目录”选择不同的文件夹即可。

2. 封装与安装 (PWA)
本应用已配置为 PWA (Progressive Web App)。
安装方法：在 Chrome 或 Edge 浏览器地址栏右侧，点击“安装应用”图标（通常是一个带加号的电脑图标），即可将其安装为独立桌面应用。

3. 快捷操作
- 双击日期：新建事项
- 拖拽灵感：将右侧灵感拖入日历即可安排日程
- 搜索：支持全文检索和聚合导出 Markdown
